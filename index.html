In 100-150 words, summarize the learn-computing activity you did with your group.

I worked with Meng on the Command Line, Shell, Terminal, Bash activity. We learned how to give commands to the computer operating system through the command line. Some useful commands include: pwd, which prints your current working directory; ls, which displays the contents of said directory; and cd, which changes to a new directory. We made a new directory labeled "test," opening an Explorer window so we could see it appear in the GUI. We then created a file named "hello.txt" with the touch command and edited it in Vim. We then viewed the new contents of the file with the cat command. We created a Python script named "hello.py" in Vim as well. After pasting some test code into the script, we ran it with the command python hello.py.

What are three projects you consider to be interesting for artistic, cultural, or other reasons?

  • I Love Bees: I mentioned ARGs before. This site was the base of a viral marketing campaign for the release of Halo 2, featuring a fictional beekeeper whose blog had been overtaken by a confused artifical intelligence. The campaign has long since reached its conclusion, but I Love Bees continues to stay up as an artifact of transmedia storytelling.
  • Every Icon: This is a web-based piece of conceptual art, consisting of a 32 x 32 square grid of either black or white squares. By progressing through combinations of black and white squares until every square is black, it will one day, trillions upon trillions of years later, have shown every possible image. If it still exists by then.
  • Sea and Spar Between: This is a random poetry generator that draws upon Emily Dickinson's poems and lines from Herman Melville's Moby Dick. It maps each of the resulting 225 trillion stanzas, comparable to the number of fish in the sea, to unique XY coordinates.
